# JS Practice

This repository is for learning and practicing JavaScript fundamentals in a way that is easy to track over time.

## Topics to Cover

- DOM manipulation
- Events
- ES6+ modern JavaScript
- Async JavaScript
- Fetch API
- Error handling
- Local storage

## How to Use This Repo

1. Add notes and examples under the topic files.
2. Update `progress.md` as you finish practice sessions.
3. Commit changes regularly so your progress is saved in Git.

## Suggested Workflow

- Pick one topic.
- Write a small example.
- Test it in the browser.
- Record what you learned.
- Commit the change.
